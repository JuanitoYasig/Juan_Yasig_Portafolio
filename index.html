"Fortify AI" - Sistema de Ciberseguridad con IA

Resumen: "Fortify AI" es una aplicación web para analizar la seguridad de dispositivos en redes locales mediante escaneos automáticos de puertos y servicios usando Nmap. El sistema valida direcciones IP, ejecuta escaneos rápidos, muestra resultados en una interfaz moderna y responsiva, y utiliza inteligencia artificial para analizar vulnerabilidades y proponer acciones de mitigación. Todo el flujo es dinámico y seguro, sin almacenar datos sensibles.

Tecnologías y lenguajes: PHP, Python, Nmap, Google Generative AI, AJAX, Bootstrap 5, FontAwesome, HTML, CSS, JavaScript

"Greenesse" - Página web informativa

Resumen: "Greenesse" es una página web informativa dedicada a promover la salud integral y el bienestar.
